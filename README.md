
## Installation
```
cd ~
git clone https://github.com/TheFuzzyGiggler/K1_Heater_Test.git
cd K1_Heater_Test
./install.sh
```
If you get a permission denied error, simply run
```
chmod +x install.sh
./install.sh
```
## Uninstall
To remove the probing patch and revert back to the original file
```
cd ~
cd K1_Heater_Test
./install.sh uninstall
```
